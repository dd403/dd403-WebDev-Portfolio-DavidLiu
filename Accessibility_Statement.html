Accessibility Statement

  • Semantic HTML Structure

    The website uses semantic HTML5 elements (header, nav, main, section, article, footer) to provide a clear document structure. This helps screen readers and other assistive technologies better understand and navigate the content.

  • Alternative Text for Images

    All images include descriptive alt text that conveys the essential information about the image content. This ensures that users with visual impairments can understand the visual content through screen readers.

  • Keyboard Navigation

    The website is fully navigable using only a keyboard. All interactive elements are accessible via keyboard controls, and the navigation menu is structured to provide a logical tab order.

  • Color Contrast

    Text and background colors have been carefully chosen to maintain a minimum contrast ratio of 4.5:1 for normal text and 3:1 for large text, ensuring readability for users with visual impairments.

  • Responsive Design

    The website is responsive and adapts to different screen sizes and devices. This ensures that content is accessible and readable on all devices, including mobile phones and tablets.

  • Text Scaling

    The website supports text scaling up to 200% without loss of content or functionality. Users can adjust text size according to their needs using browser controls.

WCAG 2.1 Compliance

This website strives to meet the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards. We are committed to making our website accessible to all users, including those with disabilities. If you encounter any accessibility issues, please contact us, and we will work to address them promptly.
