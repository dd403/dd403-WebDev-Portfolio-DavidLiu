Treemap Visualizer

Built a hierarchical tree structure representing file systems and research-paper datasets. Recursive algorithms partition the display proportionally, while interactive controls let users expand, collapse, select, and modify nodes dynamically.

  • Recursive region-partitioning algorithms
  • Interactive node expansion, collapse, and selection
  • Abstract node types using inheritance and OOP

How to Use the Python App

Controls

ClickSelect a node

Select a file or folder and display its path and size.

EExpand folder

Reveal the selected folder's immediate children.

AExpand all

Reveal every nested folder below the selected node.

CCollapse

Collapse the selected node's parent folder.

XCollapse all

Return the entire visualization to its collapsed state.

Q / BFocus / Back

Focus on a selected subtree, then return to the previous view.

↑ / ↓Resize

Increase or decrease the selected file's visualized size.

MMove

Move a selected file into the folder currently under the pointer.

DeleteRemove

Delete the selected item from the visualization.

Running locally

  1. Install Python and Pygame.
  2. Set PATH_TO_VISUALISE to the folder you want to inspect.
  3. Run python treemap_visualiser.py.
  4. Resize the window at any time; the rectangles recalculate automatically.
